Resources

Curated collection of the best free resources for Cloud and DevOps

Free Courses

  • AWS Cloud Practitioner Essentials Official AWS free course covering cloud concepts, core services, security, architecture, and pricing.
  • Microsoft Learn: Azure Fundamentals (AZ-900) Free self-paced learning path covering Azure cloud concepts, services, security, and governance.
  • Google Cloud Skills Boost: Cloud Engineer Path Hands-on labs and courses for Google Cloud fundamentals with free tier access.
  • freeCodeCamp DevOps Courses Free full-length video courses on Docker, Kubernetes, AWS, Terraform, and more.
  • KodeKloud: DevOps Pre-Requisite Course Free beginner course covering Linux, networking, applications, and infrastructure basics.
  • edX: Introduction to Linux (Linux Foundation) Free comprehensive Linux course by The Linux Foundation covering command line, file system, and administration.

Official Documentation

  • AWS Documentation Comprehensive documentation for all AWS services with tutorials and best practices.
  • Azure Documentation Microsoft's official Azure docs with quickstarts, tutorials, and architecture guidance.
  • Google Cloud Documentation GCP documentation with guides, API references, and solution architectures.
  • Kubernetes Documentation Official K8s docs with concepts, tasks, tutorials, and API reference.
  • Docker Documentation Official Docker docs covering installation, Dockerfiles, Compose, and best practices.
  • Terraform Documentation HashiCorp's official Terraform docs with provider references and configuration language guide.

Cheat Sheets

  • Git Cheat Sheet (GitHub Education) Quick reference for the most commonly used Git commands and workflows.
  • kubectl Quick Reference Official Kubernetes cheat sheet with common kubectl commands and examples.
  • Docker CLI Cheat Sheet Official Docker cheat sheet with container, image, volume, and network commands.
  • AWS Cheat Sheets (Tutorials Dojo) Concise summaries of every major AWS service for quick review and exam prep.
  • Linux Commands Cheat Sheet Essential Linux commands organized by category for quick reference.

Practice Platforms

  • AWS Free Tier 12 months of free access to core AWS services including EC2, S3, RDS, and Lambda.
  • Azure Free Account $200 credit for 30 days plus 12 months of free popular services on Azure.
  • Google Cloud Free Tier $300 credit for 90 days plus always-free tier for many GCP services.
  • Play with Docker Free browser-based Docker playground. No installation needed, 4-hour sessions.
  • KillerCoda Free interactive browser-based scenarios for Kubernetes, Docker, Linux, and more.
  • DevOps Exercises (GitHub) Massive collection of DevOps interview questions and exercises covering all major topics.

Recommended Books

  • The Phoenix Project By Gene Kim, Kevin Behr, George Spafford. A novel about IT and DevOps transformation that explains core DevOps principles through storytelling.
  • The DevOps Handbook By Gene Kim, Jez Humble, Patrick Debois. Practical guide to implementing DevOps practices in any organization.
  • Kubernetes Up & Running By Brendan Burns, Joe Beda, Kelsey Hightower. Comprehensive guide to Kubernetes from the people who created it.
  • Terraform: Up & Running By Yevgeniy Brikman. Practical guide to writing, planning, and deploying infrastructure as code with Terraform.
  • Site Reliability Engineering (Free Online) By Google. The foundational SRE book explaining how Google runs production systems. Available free online.

YouTube Channels

  • TechWorld with Nana Clear, well-structured DevOps tutorials covering Docker, Kubernetes, Terraform, CI/CD, and cloud.
  • Abhishek Veeramalla DevOps Zero to Hero series with real-world projects. Complete free DevOps course.
  • freeCodeCamp Full-length free courses on AWS, Docker, Kubernetes, Linux, Python, and much more.
  • NetworkChuck Energetic networking, Linux, cloud, and cybersecurity tutorials for beginners.
  • Kunal Kushwaha Free DevOps Bootcamp series covering Git, Docker, Kubernetes, CI/CD, and cloud native technologies.
  • Fireship Fast-paced technology explainers and "100 seconds" videos for understanding concepts quickly.
  • That DevOps Guy (Marcel Dempers) In-depth Kubernetes, Docker, and DevOps tutorials with practical, production-focused examples.
